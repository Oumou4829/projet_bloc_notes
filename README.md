# Application Bloc Notes

Application de gestion de notes permettant à l'utilisateur de :

- Créer une note  
- Lire les notes  
- Modifier une note  
- Supprimer une note  

Chaque note contient :
- Un titre  
- Un contenu  
- Une couleur  
- Une date  

## Technologies utilisées
- Flutter (Frontend)
- API REST (Backend si موجود)
