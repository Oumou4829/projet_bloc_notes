# Application Bloc Notes

Application de gestion de notes permettant à l'utilisateur de :

- Créer une note  
- Lire les notes  
- Modifier une note  
- Supprimer une note  

Chaque note contient :
- Un titre  
- Un contenu  
- Une couleur  
- Une date  

## Technologies utilisées
- Flutter (Frontend)
- Provider (Gestion d'état)

---

## 🚀 Lancement

### Commande Simple
```bash
flutter run
```

### Chrome avec Port Fixe
```bash
run-web.bat          # Windows
bash run-web.sh      # Linux/Mac
```
**➜ http://localhost:8080**

### Brave Browser
```bash
run-brave.bat        # Windows
```
**➜ http://localhost:8080**

Voir `LANCER_BRAVE.md` pour la configuration.

---

## 📱 Scripts Disponibles

| Script | Navigateur | Port |
