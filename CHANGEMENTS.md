# 🎨 Changements de Couleurs

## Modifications Effectuées

### Couleur : Bleu Ciel (Light Blue)

Tous les éléments suivants ont été changés en **bleu ciel** :

#### 1. AppBar (Barre de Navigation)
- **HomePage** : `Colors.lightBlue[400]`
- **CreateNotePage** : `Colors.lightBlue[400]`
- Texte en blanc pour un bon contraste

#### 2. Boutons
- **FloatingActionButton** (bouton +) : `Colors.lightBlue[400]`
- **ElevatedButton** (bouton Sauvegarder) : `Colors.lightBlue[400]`

#### 3. Thème Principal
- **ColorScheme** : `Colors.lightBlue`
- Appliqué dans `main.dart`
